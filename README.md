## Instalando e Rodando localmente

Clone o projeto

```bash
  git clone https://github.com/DevDaniel21/Neobyte-App.git
```

Entre no diretório do projeto

```bash
  cd Neobyte-App
```

Instale as dependências

```bash
  npm install
```

Inicie o servidor

```bash
  npm start
```
